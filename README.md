# PTAR SYSTEM

Sistema web desarrollado para la gestión y monitoreo de datos de una Planta de Tratamiento de Aguas Residuales (PTAR).

## Descripción General

La aplicación permite registrar, almacenar y visualizar parámetros físico-químicos obtenidos en los diferentes puntos de muestreo de la PTAR.

## Objetivo

Optimizar el proceso de registro y monitoreo de los parámetros de calidad del agua mediante una plataforma web centralizada.

## Módulos del Sistema

```text
PTAR SYSTEM
│
├── Inicio
│   ├── Total muestras
│   ├── Estado PTAR
│   └── Alertas
│
├── Muestreo
│   ├── Registro de muestras
│   └── Tabla de registros
│
├── Monitoreo de Parámetros
│   └── Gráficos estadísticos
│
├── Reportes
├── Usuarios
└── Acerca del Sistema
```

## Parámetros Monitoreados

- pH
- Temperatura
- Conductividad
- Turbidez
- DBO
- DQO
- SST
